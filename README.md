Данный docker-compose разворачивает раннер для гитлаба в контейнере докера.\
Для работостособности необходимо в корне создать файл # .env
```
# .env
GITLAB_TOKEN="ваш_токен_от_гитлаба_здесь"  # Замените на реальный токен
GITLAB_URL="http://192.168.1.50:8929" # Замените на реальный ip и порт
```
Чтобы поднять в фоновом режиме: docker-compose up -d.
